# *.ogm (Ogg Media)

___

## About

Video format used by the game engine

## Technical information

- No Audio
  - The audio track must be a separate [*.ogg](ogg.md) file
- Compression format: [Theora](https://en.wikipedia.org/wiki/Theora)
