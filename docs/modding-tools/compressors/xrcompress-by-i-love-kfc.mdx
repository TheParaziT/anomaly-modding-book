## About

Updated [*.db](../../references/file-formats/archives-resource-packs/db.md)/[*.xdb](../../references/file-formats/archives-resource-packs/xr.md) archiver based on xrCompress by OGSR Team

![editor centered](assets/images/xrcompress.png)

## Features

- Can be packed in any part of the game
- Ability to set your own encryption keys
